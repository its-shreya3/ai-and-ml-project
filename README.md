#  Tic Tac Toe AI

A smart game of Tic Tac Toe with an unbeatable AI opponent implemented using the **Minimax algorithm** along with **Alpha Beta Pruning**.

**Play against a perfect AI opponent who will never lose** (either wins or draws)!

---

##  Features

- **Unbeatable AI** opponent implemented using the Minimax + Alpha Beta Pruning algorithm
- Simple and clean user interface for a smooth gaming experience
- Two difficulty levels for the AI opponent:
  - **Easy** - AI makes random moves
  - **Impossible** - AI makes perfect moves
- Score tracking for wins and losses for both the player and AI opponent
- Game reset and start options
- Highlights winning combination

**Mobile Responsive** design for a smooth gaming experience

---

##  Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript** (ES6+)
- **Minimax Algorithm**
- **Alpha Beta Pruning** for optimized performance

---

##  How to Play

1. You play as **X** - You are the first player.
2. AI plays as **O**.
3. Click anywhere on the board to place your mark.
4. First to achieve **3 in a row** wins the game.
5. Game draws if all squares are filled without a winner.
